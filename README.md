# Products-Portal

Products Portal is an internal web-based application designed to help employees and new recruits understand
a company's products, services, and operational tools.

# The system provides a centralized access to:

- Product information and documentation
- Step-by-step HOW TO guides
- Internal training resources
- Application usage intructions 
- Training video uploads and playback
- Provide updated version information
- Will have access control 

The platform aims to improve onboarding, reduce training time, create a consistent knowledge-sharing
environment across the organization.
